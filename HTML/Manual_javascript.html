O que é javascript?

É uma linguagem utilizada principalmente para o auxilio de desenvolvimento de páginas de internet.

 

O que javascript faz?

         Páginas interativas ou dinâmicas, com recursos como: botões, exibições de horas, preenchimento de formulários, entre outros.

 

         Todo código javascript fica entre as tags <script> e </script>. A menos que você seja um desenvolvedor da AcliveJS, você não vai se preocupar com isso. O código base acima, das funções Ambiente() e Simulação(), já garante que todos os códigos digitados estão entre as tags <script>. Comentários são feitos utilizando duas barras //. Comentários em blocos devem vir entra /* e */.

 

Variáveis:

GLOBAIS

Declaradas (criadas) fora de uma função. As variáveis globais podem ser acessadas em qualquer parte do programa. O “var” é opcional nas variáveis gnal nas vari; condiçaixo:

E encadeado. om a instruçgrama.volvimento de plobais, mas obrigatório nas locais.

LOCAIS

Declaradas (criadas) dentro de uma função. Só podem ser utilizadas dentro da função onde foram criadas e precisa ser definida com a instrução VAR.

Ex:       var a, b, c;

                   Var a=2;

OBSERVAÇÃO:

With:

         Quando precisa manipular várias propriedades de um mesmo objeto, provavelmente prefere não ser obrigado a repetir todas as vezes a digitação do nome do objeto. A instrução “with”, permite fazer isso eliminando a necessidade de digitar o nome do objeto todas as vezes.

SINTAXE

Exemplo

 

With(<objeto>)

{

     // código aqui

}

 

 

With(Math)

{

     A=PI;

     B=ABS(x);

     C=e

}
